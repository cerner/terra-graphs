# Simple Multiline
This is simple multiline graph generated by providing timeouts for each line present in the graph.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-graphs`

## Usage

```jsx
import React from 'react';
import LineGraph from 'terra-graphs';
```
