# Supress trailing zeros

This is simple line graph supressing trailing zeros in ticks for all the axis.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-graphs`

## Usage

```jsx
import React from 'react';
import LineGraph from 'terra-graphs';
```
