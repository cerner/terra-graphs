# Timeseries Line Graph

This is a line graph with timeseries ticks in x-axis.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-graphs`

## Usage

```jsx

import LineGraph from 'terra-graphs';
```
