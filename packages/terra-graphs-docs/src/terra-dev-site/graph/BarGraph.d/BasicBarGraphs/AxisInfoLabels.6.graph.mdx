# Axis Info Labels

Axis info labels consists of bar graphs with simple bar axis info labels and stacked bar axis info labels.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm i @cerner/carbon-graphs --save-dev`

## Usage
```jsx
import Carbon from '@cerner/carbon-graphs/dist/js/carbon-graphs.js';
```
